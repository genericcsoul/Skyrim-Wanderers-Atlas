Atlas Instructions

Using the Wanderer's Atlas

Add Entries

  1. Choose Mark, Trail, or Range.
  2. Place or draw on the map.
  3. Use the map checkmark to keep it.
  4. Edit the title, category, confidence, and notes.

Find Entries

  1. Use Search, Mapped By, and category filters.
  2. Click an entry in the Atlas list to zoom to it.
  3. Hover over map entries to preview details.

Share Entries

  1. Share Atlas creates a code.
  2. Receive Atlas imports a shared code.
  3. Entries are saved in each browser.

Clean Up

  1. Undo reverses recent changes.
  2. Clean Atlas removes custom entries.
  3. The map image remains in place.
